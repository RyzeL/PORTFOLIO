임의 서버 연결 설명
  • 로그아웃 클릭시 새창에 로그인 페이지가 나타납니다.
  • 6개의 메뉴를 sub1, sub2, sub3...의 스타일로 파일명을 통일하였습니다.
  • 알림정보(sub1), 외주운영관리(sub2), 통합결제정보(sub4), 고객분석(sub5)에 관한 페이지를 퍼블 담당했습니다.
    그 메뉴마다의 각 3depth의 첫번째 메뉴만을 대표로 연결했습니다.
JavaScript 설명
  • html에 명시한 조건부주석에 따른 스크립트 파일 (ie9 미만에 대한 설정)
    • prefixfree.min.js : 각 브라우저 환경에 따라 다르게 동작하는 CSS3 속성을 잡아주기 위한 vender prefix를 스트립트를 사용하여 설정 상황에 따라 적용이 되지 않을 수 있는 스타일에는 각 브라우저에서 판독이 가능한 접두어를 붙여 인식할 수 있게 했습니다.
    • modernizr.custom.js : HTML5와 CSS3가 지원되지 않는 브라우저에서 작동하게 하거나 동작 않는 CSS속성을 대체 할 수 있는 다른 형식으로 표현해 주는 기능입니다.
    • html5shiv.min.js : HTML5의 시멘틱태그가 ie8에서 작동 가능하게 합니다.
  • default.js : 해당 프로젝트에 맞춘 스크립트 설정
    • 1::: a태그 클릭시 스크롤 최상위로 올라가는 이벤트 방지
      'cursor:pointer;' 또는 링크 연결을 위해 임의로 a태그에 '#' 표시한 부분을 클릭 시, 상위로 올라가는 이벤트를 방지 하기 위해 preventDefault(); 사용했습니다.
    • 2::: 전체 메뉴 구현
      jQuery show(),와 hide() 메소드 사용했습니다. 클래스명 btn_allMenu란 햄버거 버튼 클릭 시 전체 메뉴가 나타나고, 클래스명 btn_close란 닫기 버튼 클릭 시 전체 메뉴가 사라집니다.
    • 3::: 로그인 폼 정 가운데 구현
      로그인 폼 위치가 해상도에 상관없이 정 가운데로 구현되어야 하기 때문에 해당 폼의 width값과 height값의 변수와 document의 width값과 height값의 변수를 선언하여 조건식과 CSS의 스타일 값을 적용하여 구현했습니다.
    • 4::: 서브페이지 사이드바 Tab구현
      addClass()와 removeClass()를 사용하여 좌측 사이드 바 메뉴 클릭시 나타나야하는 스타일을 설정한 클래스를 추가 또는 제거하여 스타일 구현하였습니다. each() 반복문함수를 사용하여 클릭한 인덱스를 저장해둔 변수와 반복문 each 인덱스의 수가 일치할 경우를 이용하여 클래스 추가 또는 제거했습니다.
    • 5::: class="slidebarMenuBox" 높이 자동 조절
      클래스명 sidebarMenuBox에 border-right와 border-left값을 해상도마다 맞게 주기 위해 sidebarMenuBox를 제외한 나머지 height값을 변수로 저장하여 sidebarMenuBox의 height값을 설정하였습니다.
    • 6::: 팝업 정 가운데 구현
      레이어팝업의 위치를 정 가운데로 정렬하기 위해 변수 선언 및 조건식 사용하였습니다. 로그인폼 정렬과 흡사합니다.

임의 서버 연결 설명

JavaScript 설명
  • - html에 명시한 조건부주석에 따른 스크립트 파일 (ie9 미만에 대한 설정)

  • - default.js : 해당 프로젝트에 맞춘 스크립트 설정

      'cursor:pointer;' 또는 링크 연결을 위해 임의로 a태그에 '#' 표시한 부분을 클릭 시, 상위로 올라가는 이벤트를 방지 하기 위해 preventDefault(); 사용했습니다.
    • 2::: 레이어팝업 정 가운데 정렬 및 이벤트 구현
      해상도에 상관없이 레이어팝업이 정 가운데로 구현되어야 하기에 해당 폼의 width값과 height값의 변수와 document의 width값과 height값의 변수를 선언하여 조건식과 CSS의 스타일 값을 적용하여 구현했습니다.
    • 3::: Tab기능 구현
      클릭한 tab버튼 인덱스와 each문으로 돌린 tab버튼 인덱스가 맞지 않으면 해당 텝에 추가될 클래스를 제거, 맞을 경우엔 클래스를 추가하여 스타일을 구현하였습니다. 또한 클릭한 tab버튼 인덱스와 tab버튼 하단에 위치한 컨텐츠의 인덱스가 일치할 경우 해당 컨텐츠가 나타나게 하였습니다.
    • 4-1::: 레이어탭의 레이어 설정 구성항목 타이틀 클릭시 css변경 및 서브내용 구현
      가상 클래스closeFlag와 openFlag를 설정하여구현했습니다. closeFlag가 있을시, 삭제후 openFlag를 추가한 후 해당 서브내용이 3초동안 슬라이드 다운되게 하였고 반대인 경우 openFlag를 삭제후 closeFlag를 추가한 후 해당 서브 내용이 3초 동안 슬라이드 업되게 하였습니다.
    • 4-5::: 각 Tab마다 높이 자동 인식하여 스크롤 자동 구현
      클래스명'monitorBox' ,'searchTableBox1'이 컨텐츠의 길이에 따라 스크롤을 구현하기위해 나머지 높이들을 변수에 저장, 그들의 합 또한 변수에 저장 한 후 document높이를 저장한 변수에서 차액을 구하여 구현하였습니다.
    • 6::: 레이어팝업 구현
      클래스명 'popupShowBtn'를 클릭시 해당 레이어팝업이 나타나도록 인덱스를 구하여 차례로 구현하였습니다.
